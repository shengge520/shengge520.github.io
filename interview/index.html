# Vue,React,Anjular三者之间的区别？

  1. 数据流:Vue,Anjular双向数据流，React单向数据流

  2. Vue,React 共同点，都有虚拟DOM。

2.1 Vue的虚拟DOM与React的虚拟DOM区别？ virtual DOM(vDOM)不一样,vue会跟踪每一个组件的依赖关系, 不需要重新渲染整个组件树。

而对于React而言,每当应用的状态被改变时,全部组件都会重新渲染,所以react中会需要shouldComponentUpdate这个生命周期函数方法来进行控制
